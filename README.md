# mov

## install
```
# main
$ pip install git+https://github.com/EstherCho-7/movie.git

# branch
$ pip install git+https://github.com/EstherCho-7/movie.git@<BRANCH NAME>
```

## Start
```
$ git clone https://github.com/EstherCho-7/movie.git
$ cd <DIR>

$ # option
$ pdm venv create
$ source .venv/bin/activate
```
